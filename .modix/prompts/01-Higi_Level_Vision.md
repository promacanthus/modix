# High Level Vision

Agent 可能会反馈一份结构化的规格草案：概述、功能列表、技术栈建议、数据模型等。这份规格随后成为你和 Agent 都可以回溯的"单一事实来源（single source of truth）

## English

You are an AI software engineer. Draft a detailed specification for [Project X] covering objectives, features, constraints, and a step-by-step plan.

---
example:

Build a web app where users can track tasks (todo list), with user accounts, a database, a simple UI.

## 中文

你是一名 AI 软件工程师。请为 [X 项目] 起草一份详细规格说明，涵盖目标、功能、约束条件和分步计划。

---
示例：

构建一个用户可以跟踪任务（待办事项）的 Web 应用，包含用户账户、数据库和简洁的 UI

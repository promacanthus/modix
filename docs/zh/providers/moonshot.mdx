## Kimi

### 概述

Moonshot AI 提供具有高级推理能力的 Kimi 模型。

### 获取 API 访问权限

#### 1. 注册账号

- 访问 [Moonshot AI](https://www.moonshot.cn/)
- 创建账号
- 完成验证过程

#### 2. 获取 API 访问权限

- 申请 API 访问权限
- 等待批准
- 获取 API 凭据

### 配置

#### 环境变量

```bash
export MOONSHOT_API_KEY="您的-api-key-这里"
```

#### Modix 配置

```json
{
  "vendor": "moonshot",
  "company": "Moonshot AI",
  "api_endpoint": "https://api.moonshot.cn/anthropic",
  "api_key": "您的-kimi-api-key",
  "models": [
    "kimi-k2-thinking-turbo"
  ]
}
```

### 模型详情

- `kimi-k2-thinking-turbo`: 具有思考能力的高级推理

## KAT-Coder

### 概述

快手的 KAT-Coder 针对软件开发任务进行了优化。

### 获取 API 访问权限

#### 1. 注册账号

- 访问 [快手 AI 平台](https://streamlake.com/)
- 注册访问
- 完成设置过程

#### 2. 获取凭据

- 导航到 API 设置
- 创建 API 凭据
- 记下端点和密钥信息

### 配置

#### 环境变量

```bash
export KUAISHOU_API_KEY="您的-api-key-这里"
```

#### Modix 配置

```json
{
  "vendor": "streamlake",
  "company": "Kuaishou",
  "api_endpoint": "https://wanqing.streamlakeapi.com/api/gateway/v1/endpoints/ep-xxx-xxx/claude-code-proxy",
  "api_key": "您的-kat-api-key",
  "models": [
    "KAT-Coder"
  ]
}
```

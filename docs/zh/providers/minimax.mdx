## MiniMax

### 概述

MiniMax 提供在各种任务中具有强大性能的先进 AI 模型。

### 获取 API 访问权限

#### 1. 注册账号

- 访问 [MiniMax 平台](https://www.minimaxi.com/)
- 创建账号
- 完成验证

#### 2. 获取 API 访问权限

- 申请 API 访问权限
- 查看条款和条件
- 获取 API 凭据

### 配置

#### 环境变量

```bash
export MINIMAX_API_KEY="您的-api-key-这里"
```

#### Modix 配置

```json
{
  "vendor": "minimax",
  "company": "MiniMax",
  "api_endpoint": "https://api.minimaxi.com/anthropic",
  "api_key": "您的-minimax-api-key",
  "models": [
    "MiniMax-M2"
  ]
}
```

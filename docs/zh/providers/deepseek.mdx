## DeepSeek

### 概述

DeepSeek 提供高性能且价格具有竞争力的先进 AI 模型。

### 获取 API 访问权限

#### 1. 注册账号

- 访问 [DeepSeek 控制台](https://platform.deepseek.com/)
- 如果没有账号，请创建一个
- 完成注册过程

#### 2. 获取 API 密钥

- 登录您的 DeepSeek 账号
- 导航到 API 密钥部分
- 创建新的 API 密钥
- 安全地复制密钥

### 配置

#### 环境变量

```bash
export DEEPSEEK_API_KEY="您的-api-key-这里"
```

#### Modix 配置

```json
{
  "vendor": "deepseek",
  "api_endpoint": "https://api.deepseek.com/v1",
  "api_key": "您的-deepseek-api-key",
  "models": [
    "deepseek-reasoner",
    "deepseek-chat"
  ]
}
```

### 模型详情

| 模型 | 描述 | 定价 |
|------|------|------|
| `deepseek-reasoner` | 高级推理模型 | 查看网站 |
| `deepseek-chat` | 通用对话模型 | 查看网站 |

### 使用示例

```bash
# 设置 API 密钥环境变量
export DEEPSEEK_API_KEY="您的-api-key"

# 与 Modix 一起使用
modix switch deepseek-reasoner
```

### 速率限制和配额

- 在 DeepSeek 控制台中查看当前的速率限制
- 通过仪表板监控使用情况
- 如需更高限制，请联系支持

## GLM

### 概述

ZHIPU AI 的 GLM 模型提供强大的语言理解和生成能力。

### 获取 API 访问权限

#### 1. 注册账号

- 访问 [ZHIPU AI](https://open.bigmodel.cn/)
- 创建账号
- 完成注册

#### 2. 访问 API

- 导航到 API 管理
- 创建 API 凭据
- 记下端点信息

### 配置

#### 环境变量

```bash
export ZHIPU_API_KEY="您的-api-key-这里"
```

#### Modix 配置

```json
{
  "vendor": "bigmodel",
  "company": "ZHIPU AI",
  "api_endpoint": "https://open.bigmodel.cn/api/anthropic",
  "api_key": "您的-zhipu-api-key",
  "models": [
    "GLM-4.6"
  ]
}
```

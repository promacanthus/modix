## 快速开始

<Steps>

<Step title="下载并安装 Modix">
  选择以下任一方式安装 Modix：

  **方式一：从源码构建（推荐）**

  <Note> 
    **先决条件**: 您的系统必须安装 [Rust 和 Cargo](https://rustup.rs/)。 
  </Note>

  ```bash
  git clone https://github.com/promacanthus/modix.git
  cd modix
  cargo build --release
  ```

  构建后，二进制文件将位于 `target/release/modix`。

  **方式二：预构建二进制文件**

  第一个稳定版本发布后将提供预构建的二进制文件。

  <Check> 
    验证安装：运行 `modix --version` 确认安装成功。
  </Check>
</Step>

<Step title="初始化配置">
  运行初始化命令创建默认配置文件：

  ```bash
  modix init
  ```

  此命令将在 `~/.modix/settings.json`（或 Windows 上的 `%APPDATA%\modix\settings.json`）创建默认配置。

  <Check>
    配置文件已创建：请勿手动编辑配置文件，使用命令添加 API 密钥并启用模型。
  </Check>
</Step>

<Step title="查看当前模型状态">
  运行状态命令查看当前正在使用的模型和配置：

  ```bash
  modix status
  ```

  这将显示当前模型、提供商、API 端点和配置状态。

  <Check>
    当前模型信息已显示：确认模型状态正常。
  </Check>
</Step>

</Steps>

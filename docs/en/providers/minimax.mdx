## MiniMax

### Overview

MiniMax provides advanced AI models with strong performance in various tasks.

### Getting API Access

#### 1. Sign Up

- Visit [MiniMax Platform](https://www.minimaxi.com/)
- Create an account
- Complete verification

#### 2. Get API Access

- Apply for API access
- Review terms and conditions
- Obtain API credentials

### Configuration

#### Environment Variables

```bash
export MINIMAX_API_KEY="your-api-key-here"
```

#### Modix Configuration

```json
{
  "vendor": "minimax",
  "company": "MiniMax",
  "api_endpoint": "https://api.minimaxi.com/anthropic",
  "api_key": "your-minimax-api-key",
  "models": [
    "MiniMax-M2"
  ]
}
```

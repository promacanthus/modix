## ZHIPU AI GLM (Zhinao Qingyan)

### Overview

ZHIPU AI's GLM models offer powerful language understanding and generation capabilities.

### Getting API Access

#### 1. Sign Up

- Visit [ZHIPU AI](https://open.bigmodel.cn/)
- Create an account
- Complete registration

#### 2. Access API

- Navigate to API management
- Create API credentials
- Note the endpoint information

### Configuration

#### Environment Variables

```bash
export ZHIPU_API_KEY="your-api-key-here"
```

#### Modix Configuration

```json
{
  "vendor": "bigmodel",
  "company": "ZHIPU AI",
  "api_endpoint": "https://open.bigmodel.cn/api/anthropic",
  "api_key": "your-zhipu-api-key",
  "models": [
    "GLM-4.6"
  ]
}
```

## Doubao

### Overview

ByteDance's Doubao offers AI models optimized for various applications including coding.

### Getting API Access

#### 1. Sign Up

- Visit [ByteDance Cloud](https://cloud.volces.com/)
- Register for an account
- Complete identity verification

#### 2. Access API

- Navigate to the Doubao service
- Enable API access
- Create API credentials

### Configuration

#### Environment Variables

```bash
export BYTEDANCE_API_KEY="your-api-key-here"
export BYTEDANCE_API_SECRET="your-api-secret-here"
```

#### Modix Configuration

```json
{
  "vendor": "volcengine",
  "company": "ByteDance",
  "api_endpoint": "https://ark.cn-beijing.volces.com/api/coding",
  "api_key": "your-doubao-api-key",
  "models": [
    "doubao-seed-code-preview-latest"
  ]
}
```

### Usage Notes

- Ensure proper network access to ByteDance services
- Check regional availability and endpoints
- Monitor usage through the ByteDance console

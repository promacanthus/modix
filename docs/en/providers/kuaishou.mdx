## KAT-Coder

### Overview

Kuaishou's KAT-Coder is optimized for software development tasks.

### Getting API Access

#### 1. Sign Up

- Visit [Kuaishou AI Platform](https://streamlake.com/)
- Register for access
- Complete setup process

#### 2. Obtain Credentials

- Navigate to API settings
- Create API credentials
- Note the endpoint and key information

### Configuration

#### Environment Variables

```bash
export KUAISHOU_API_KEY="your-api-key-here"
```

#### Modix Configuration

```json
{
  "vendor": "streamlake",
  "company": "Kuaishou",
  "api_endpoint": "https://wanqing.streamlakeapi.com/api/gateway/v1/endpoints/ep-xxx-xxx/claude-code-proxy",
  "api_key": "your-kat-api-key",
  "models": [
    "KAT-Coder"
  ]
}
```

## Qwen (Tongyi Qianwen)

### Overview

Alibaba's Qwen models offer powerful AI capabilities with Chinese language optimization.

### Getting API Access

#### 1. Sign Up

- Visit [Alibaba Cloud](https://www.alibabacloud.com/)
- Create an account or log in
- Navigate to the Qwen (Tongyi Qianwen) service

#### 2. Enable Service

- Find the Qwen service in the console
- Enable the service for your account
- Complete any verification steps

#### 3. Obtain API Credentials

- Go to the API Keys section
- Create a new API key
- Note the API key and secret

### Configuration

#### Environment Variables

```bash
export ALIBABA_API_KEY="your-api-key-here"
export ALIBABA_API_SECRET="your-api-secret-here"
```

#### Modix Configuration

```json
{
  "vendor": "bailian",
  "company": "Alibaba",
  "api_endpoint": "https://dashscope.aliyuncs.com/compatible-mode/v1",
  "api_key": "your-qwen-api-key",
  "models": [
    "qwen3-coder-plus",
    "qwen3-coder-flash"
  ]
}
```

### Model Details

| Model | Description | Use Case |
|-------|-------------|----------|
| `qwen3-coder-plus` | Advanced coding model | Software development |
| `qwen3-coder-flash` | Fast coding model | Quick code generation |

### Usage Examples

```bash
# Set environment variables
export ALIBABA_API_KEY="your-api-key"
export ALIBABA_API_SECRET="your-secret"

# Configure Modix
modix switch qwen3-coder-plus
```

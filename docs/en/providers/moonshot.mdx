## Moonshot AI Kimi

### Overview

Moonshot AI provides the Kimi models with advanced reasoning capabilities.

### Getting API Access

#### 1. Sign Up

- Visit [Moonshot AI](https://www.moonshot.cn/)
- Create an account
- Complete verification process

#### 2. Get API Access

- Apply for API access
- Wait for approval
- Obtain API credentials

### Configuration

#### Environment Variables

```bash
export MOONSHOT_API_KEY="your-api-key-here"
```

#### Modix Configuration

```json
{
  "vendor": "moonshot",
  "company": "Moonshot AI",
  "api_endpoint": "https://api.moonshot.cn/anthropic",
  "api_key": "your-kimi-api-key",
  "models": [
    "kimi-k2-thinking-turbo"
  ]
}
```

### Model Details

- `kimi-k2-thinking-turbo`: Advanced reasoning with thinking capabilities

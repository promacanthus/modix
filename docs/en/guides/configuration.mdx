## Configuration File Details

Modix stores its configuration in `~/.modix/settings.json` (or `%APPDATA%\modix\settings.json` on Windows).

### Configuration Structure

The configuration file uses a nested structure to organize models by vendor:

```json

  "current_vendor": "anthropic",
  "current_model": "Claude",
  "default_vendor": "anthropic",
  "default_model": "Claude",

      "company": "Anthropic",
      "api_endpoint": "",
      "api_key": "",

        "Claude"
      ]
    },

      "company": "DeepSeek",

```

Key changes:

- **Vendor-Based Organization**: Models are now organized under vendors in the `vendors` object
- **Array-Based Models**: Each vendor has a `models` array containing all model names
- **Shared Credentials**: API endpoints and keys are shared across all models within a vendor
- **Current/Default Tracking**: Tracks the currently selected and default vendor and model

<Info>
  **Prerequisites**:
  - Rust development environment
  - Cargo package manager
</Info>

Follow these steps to develop and run Modix on your system.

## Development

### Project Structure

```bash
src/
├── main.rs           # CLI entry point
├── lib.rs            # Library exports
├── config.rs         # Configuration data structures
└── config_manager.rs # Configuration management logic
```

### Build

```bash
cargo build
```

### Run Tests

```bash
cargo test
```

### Run with Arguments

```bash
cargo run -- list
cargo run -- switch claude
```

### Code Formatting

```bash
cargo fmt
```

### Linting

```bash
cargo clippy
```
